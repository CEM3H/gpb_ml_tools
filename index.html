Добро пожаловать в документацию GPB Library!

GPB Library - это библиотека для разработки и внедрения моделей машинного обучения, которая предоставляет следующие возможности:

  • Работа с данными

    • Загрузка данных из различных источников

    • Валидация данных и схем

    • Предобработка данных

    • Отбор признаков

  • Работа с моделями

    • Обучение моделей классификации и регрессии

    • Оптимизация гиперпараметров

    • Калибровка вероятностей

    • Оценка качества моделей

  • Отчетность и визуализация

    • Генерация отчетов в формате MS Word

    • Визуализация метрик и результатов

    • Построение графиков и диаграмм

    • Анализ важности признаков

  • Инфраструктура

    • Контейнеры для хранения моделей и артефактов

    • Пайплайны для автоматизации процессов

    • Сериализация и десериализация объектов

    • Управление метаданными
